
# Spotify Clone — Minimal Web Music Player

A lightweight, Spotify-like web player built with plain HTML, CSS and JavaScript. It demonstrates a simple audio UI with a playlist, cover art, playback controls and a seek bar — great for learning DOM manipulation and the HTML5 Audio API.

Key features
- Play / pause the selected track
- Next / previous track navigation
- Seek using the progress bar
- Display cover art and current track name

Files of interest
- `index.html` — app markup
- `style.css` — styling and layout
- `script.js` — player logic (playlist, audio control, UI updates)
- `songs/` — example MP3 files used by the player
- `covers/` — cover images for each song

1. Open `index.html` directly in your browser (double-click or use your editor's Live Preview).

Customization
- Add or replace audio files in the `songs/` folder and cover images in `covers/`.
- Update the `songs` array in `script.js` to change song names, file paths, or cover paths.

Browser support
- Works in modern browsers (Chrome, Firefox, Edge, Safari). For local assets, serving over HTTP avoids potential file:// restrictions.
